3. Вы проектируете базу данных поликлиники. Помимо прочего, у вас есть таблица лекарственных препаратов и таблица пациентов.
Врач может назначать препараты пациентам и это запоминается в базе. Какой вид связи будет между этими двумя таблицами?

4. Каким общим словом называются базы данных, содержащие таблицы, состоящие из рядов однотипных данных?
A: реляционные

5. Что такое ACID в контексте реляционных баз данных?
A: Атомарность, Согласованность, Изолированность, Устойчивость
